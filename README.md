# lr

a .bundle/.morlru Blue Archive extractor made with tauri + svelte

features  
- extract .bundle files  
- extract .morlru files  
- very lightweight desktop app  
- built with tauri and svelte  

# usage  
1. download from the [release](https://github.com/ThatWasMinimal/molru-extractor/releases)
2. import a .morlru into lr
3. extract it
